# Planner de Estudos

Projeto desenvolvido com HTML, CSS e JavaScript para gerenciamento de tarefas de estudo.

## Funcionalidades

- Adicionar tarefas
- Excluir tarefas
- Marcar tarefas como concluídas
- Filtrar tarefas
- Contadores de tarefas
- Persistência de dados com LocalStorage

## Tecnologias

- HTML5
- CSS3
- JavaScript
